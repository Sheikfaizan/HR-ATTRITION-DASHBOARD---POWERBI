# HR-ATTRITION-DASHBOARD-POWERBI
The main objective of this project is to analyze hospital data to understand patient trends, treatment performance, and hospital efficiency. The dashboard helps in making data-driven decisions by providing clear insights into patient admissions, discharge rates, and overall hospital performance.

# DATASET

The dataset contains hospital-related information such as:

 Patient ID

 Patient Name

 Patient Gender

 Patient Age

 Patient Admin Date

 Patient Race

 Patient Wait time

 Department Referral

 Admission Status

 AgeGroup

This data was used to identify patterns and trends in hospital operations.

# TOOLSANDTECHNOLOGIESUSED

 PowerBI–forcreating interactive dashboards

 Microsoft Excel / CSV– for storing and preparing the dataset

 PowerQuery–for data cleaning and transformation

 DAX(DataAnalysis Expressions)– for creating calculated measures and KPIs

# METHODOLOGY
1. Collected and imported hospital dataset into Power BI.
2. Cleaned and transformed the data using Power Query.
3. Created relationships between tables if needed.
4. Developed calculated measures using DAX.
5. Designed interactive visualizations such as charts, tables, and KPIs.
6. Built a dashboard to present insights clearly.

# IMPLEMENTATION
The dashboard was implemented in Power BI with multiple visual components such as:

 Patient admission trends

 Department-wise patient count

 Average length of hospital stay

 Treatment cost analysis

 Doctor performance overview

Filters and slicers were added to allow users to explore the data interactively.

# OUTCOME
The final outcome is an interactive Hospital Performance Dashboard that provides
valuable insights into patient flow, treatment costs, and departmental performance. It helps
stakeholders quickly analyze hospital operations and make better decisions.
